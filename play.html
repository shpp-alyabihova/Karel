Karel Commands Help

Built-in Karel commands:

move(); - move robot one step to current direction

turnLeft(); - turn current robot direction to left

turnRight(); - turn current robot direction to right

putBeeper(); - take beeper from the bag and put it to the cell

pickBeeper(); - take beeper from the cell and put into the bag

Karel conditions

frontIsClear() / frontIsBlocked() - check for a wall in front

leftIsClear() / leftIsBlocked() - check for a wall from left

rightIsClear() / rightIsBlocked() - check for a wall from right

beepersPresent() / noBeepersPresent() - check for beepers in cell

beepersInBag() / noBeepersInBag() - check for beepers in bag

facingNorth() / notFacingNorth() - check current direction. Is it North?

facingEast() / notFacingEast() - check current direction. Is it East?

facingSouth() / notFacingSouth() - check current direction. Is it South?

facingWest() / notFacingWest() - check current direction. Is it West?

Karel Code Editor Help

While it is based on CodeMirror Ace editor most of usual programming shurtcuts are supported. It is used keymap of Sublime Text key binds

Karel keybindings

There are some keybindings, provided by KarelOnline team, to help you with maintaining you code

Use Ctrl-Enter to execute code.

To manage font size of code-editor use Ctrl-PgUp and Ctrl+PgDn
