Simple play with Karel and learn programming
Karel is very simple for programming.
He can move, turn and pick up or put down beepers.
You can ask: what are the beepers?
And we answer: don't know!
May be something, that can make sound "beep".
May by not:)
He know only few short commands:
move();
turnLeft();
putBeeper();
pickBeeper();

But if you want more commands - you can create own!
function turnAround() {
turnLeft();
turnLeft();
}
and you can use new coomand turnAround() is your code!

Using that you can solve many tasks.
You can ask Karel about the world around him.
He understand query like this:
About the walls: is the
frontIsClear() or frontIsBlocked() ?
leftIsClear() or leftIsBlocked() ?
rightIsClear() or rightIsBlocked() ?
About the beepers: are the
beepersPresent() or noBeepersPresent() ?
beepersInBag() or noBeepersInBag() ?
And about it direction: are you
facingNorth() or notFacingNorth() ?
facingEast() or notFacingEast() ?
facingSouth() or notFacingSouth() ?
facingWest() or notFacingWest() ?
But Karel programming impossible without conditions and loops!
if ( condition ) {
statements executed if condition is true
}

if ( condition ) {
statements executed if condition is true
} else {
statements executed if condition is false
}

Example of using such code:
if ( frontIsBlocked() ) {
move();
} else {
turnAround();
}

And iterative statements:
for ( int i = 1; i < count; i++ ) {
statements to be repeated
}

while ( condition ) {
statements to be repeated
}
Program's for Karel very short and look's like this:

while ( frontIsClear() ) {
move();
putBeeper();
}

or
for (var i=0; i < 10; i++ ) {
move();
if ( beepersPresent() ) {
pickBeeper();
}
}
Using this, you'll be able to program any robot very soon:)

After you solve school tasks you can create own on map-editor page.

That's all! Try it and fall in love to Karel with us!
